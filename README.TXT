PROJECT TITLE: Stick Game MP
PURPOSE OF PROJECT: Create a networked 2d wave defense game for 1-4 players.
VERSION/DATE: v1.0 8/12/2018
HOW TO START THIS PROJECT: Open package with BlueJ, right click on "level" and 'run static void main'.
AUTHORS: MEHS CS CLUB
HOW TO PLAY: WASD TO MOVE, SPACEBAR TO SHOOT
